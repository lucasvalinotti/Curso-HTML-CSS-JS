ESPECIFICIDAD

Los estilos siguen una "jerarquia" de acuerdo a si se los aplica por id, clase o el nombre del elemento.
Orden de especificidad:

  1. Etiqueta !important en CSS (EVITAR)
  2. Estilo en linea de HTML
  3. ID
  4. Clase
  5. Elementos

"Medidas" de Especificidad

  1. (1,0,0)
  2. (0,1,0)
  3. (0,0,1)
  4. (0,0,0)

Puedo aumentar la especificidad "sumando" dos selectores
Por ejemplo: #id .clase tendra una especificidad (1,1,0) porque es (1,0,0) + (0,1,0)
